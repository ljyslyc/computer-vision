Introduction

Images are made up of many different frequencies. Isolating and manipulating these frequencies can enhance features of an image, create illusions, blend images together, and much more. For this assignment we were tasked to implement many of these techniques and produce interesting images of our own.

Part 0. Warmup

As a warmup the first technique we were asked to implement was sharpening an image using unsharpening. The technique is called unsharpening because it uses a blurred version of the image to acquire the high frequency edges needed to add back to the original image. First the original image is convolved with a 2d gaussian function, with a sigma value of 40, to get the low frequencies of the image a.k.a. the blurred image. Next the high frequencies are gotten by subtracting the low frequency image from the original image. Finally to sharpen the image a percentage of the high frequency image is added back to the original image.

sharpened_img = original_img + percent*(original_img - blurred_img)

Part 1. Hybrid Images

A hybrid image is one that can be perceived in two different ways depending on how you view it. Because of how the human eye processes frequencies, low frequencies are observed over high frequencies at larger distances from an object and high frequencies are observed over low frequencies at closer distances from an object. This allows for the illusion of a changing image to be created by combining the low frequencies of one image and the high frequencies of another. For this hybrid image the viewer will see one image while looking at the scene from a distance and a different image while looking at the scene close up.

Part 2. Gaussian and Laplacian Stacks

Gaussian and Laplacian Stacks can be used to analyze images that contain structure at multiple resolutions, such as the hybrid images created above. A Gaussian stack is a stack of the same image convolved with a 2d gaussian that gets larger at each level. To create a larger gaussian filter, a larger sigma value is used. For each level of the Gaussian stack that I created the sigma value was doubled. To create a Laplacian stack the second level of the gaussian stack is subtracted from the first to make the first level Laplacian and then the third level gaussian is subtracted from the second level to make the second level Laplacian and so on. The lowest level of the Gaussian stack is also the lowest level of the Laplacian stack. V: Multiresolution Blending

The most common technique for blending two images together is alpha blending. Alpha blending is the process of combining an image with a background to create the appearance of partial or full transparency. Two images can be combined by using a black and white mask where one image will alpha blend with the black and the other will blend with the white, resulting with a final image that fades from one of the original images to the other. However, the optimal alpha blending window (the width of the feathering along the seams of the mask) is hard to achieve when blending two images together using all of the frequencies at once. This is because windows that are too small cause abrupt seams between the images because they don’t blend the lower frequencies enough, but windows that are too larger cause shadowing in the image where high frequencies begin to overlap in the final image. To fix these problems the two images are decomposed into Laplacian stacks to separate the different frequency layers. The mask image is decomposed into a Gaussian stack to increase the blending window at lower layers. Then, at each layer both images are alpha blended with the mask for that layer and added together. After this all of the alpha blended layers are summed to get the final result. For the resulting images I used Gaussian and Laplacian stacks 3 layers deep. For the mask's Gaussian stack I introduced another parameter "alpha" which I used to increase the size of the Gaussian filter at each level more then in the Gaussian stack I implemented earlier. This is because for some images I wanted the lower frequencies to have a very large window while for others I wanted a small one. In the earlier Gaussian stack the sigma values were calculated by:

sigma = 2**x

Where x is the current layer of the stack.

In the alpha Gaussian Stack, the sigma values were calculated by:

sigma = 2**(x*alpha)
